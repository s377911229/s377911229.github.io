发表于 2020-08-17

建立博客

  1. 建立仓库username.github.io

  2. 安装并配置git，nodejs

  3. 本地安装及使用

    npm install -g hexo

    cd blog

    hexo init

    npm install

    npm install hexo-deployer-git –save

    hexo g

    hexo s

  4. 增加主题、密码和git

    git clone https://github.com/theme-next/hexo-theme-next themes/next

    npm install –save hexo-blog-encrypt

    修改_config.yml：

    ​ language: zh-CN

    ​ theme: next

    ​ deploy:

      type: git
  repository: git@github.com:s377911229/s377911229.github.io.git
  branch: master

    ​ encrypt:

      enable: true

    修改blog.md：

    ​ password: 123456

    ​ message: 输入密码

    hexo clean

    hexo g

    hexo d
